# Mila 1.9.0 (beta 1)

This is a **beta** build — it has everything we've built since 1.8.14 but may be a little rougher than a stable release. Thanks for testing! You can switch back anytime by turning off **Enable beta version** in Settings → Updates.

It's a big one. Highlights since 1.8.14:

## New

- **Transcribe on a remote server.** Point Mila at any OpenAI-compatible API — OpenAI's Whisper, or your own self-hosted model like ivrit.ai — instead of transcribing on-device. Set it up in Settings → Models.
- **iPhone Voice Memos sync.** Mila can import and auto-transcribe the Voice Memos that iCloud syncs from your iPhone. Pick which folders to watch, set a start date so it doesn't backfill years of old recordings, and it files them under a dedicated **Voice Memos** folder. It only needs access to that one folder — no Full Disk Access.
- **Opt into betas.** A new Settings → Updates tab lets you enable pre-release builds (that's how you got this one).

## Transcription quality

- **Fewer hallucinations.** A neural voice-activity detector skips non-speech, so Whisper stops inventing filler text on silence or background noise.
- **Auto-discard accidental clips.** Very short recordings with no speech — hotkey misfires, empty captures — are dropped automatically. Tune the threshold (or turn it off) in Settings → Storage.

## Queue & recordings

- **Stop or remove Queue items.** Cancel a stuck or unwanted transcription right from the Queue. Stopped items move to Recently Deleted instead of showing up as "Failed."
- **No more stuck "Queued" items** after you quit and relaunch mid-transcription.
- **A color per speaker** when speaker labels are on, so who-said-what is easier to scan.
- **Export SRT subtitles mid-recording** from the live transcript.
- Collapsible "All Transcriptions" section in the sidebar.

## AI & summaries

- **Automatic summaries** after each recording (toggle in Settings → LLM), a configurable CLI timeout, and a Test panel to debug your LLM setup.

## Fixes & reliability

- A large bug-fix sweep across the transcription engine, audio capture, and UI, with new regression tests.
- Fixed the stray border around the dictation pill, restored the sidebar's right-click menu, and many smaller fixes.
- Added CodeQL security scanning to every build.
